# SEED_MEANapp
This is an App Seed utilizing Mongdb Express Angular Node and Materialize.css by ZIMPLE.IO.
## Requirements
* Node.js
* NPM

## Getting Started
Run the following command in the root of the project.
```
  npm run setup
```
## Generators
Enter Developement Mode
```
  npm run dev_mode
```
Create rest api
```
  npm run create_api
```
Create a view
```
  npm run create_view
```
Create a component
```
  npm run create_componenet
```
